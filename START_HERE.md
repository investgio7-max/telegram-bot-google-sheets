## ⚡ 3 шага до работающего бота

### Шаг 1: Telegram Bot
```
1. Найти @BotFather в Telegram
2. Написать: /newbot
3. Получить BOT_TOKEN
```

### Шаг 2: Google Sheets Setup
```
1. Создать Service Account в Google Cloud
2. Скачать credentials.json
3. Создать Google Sheets таблицу
4. Выдать доступ сервис-аккаунту
```

### Шаг 3: Запуск
```
1. pip install -r requirements.txt
2. Создать .env с BOT_TOKEN и SPREADSHEET_ID
3. python bot.py
```

**Готово! Бот работает! 🎉**
