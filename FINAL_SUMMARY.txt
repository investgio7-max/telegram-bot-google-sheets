================================================================================
                          БЫСТРЫЙ СТАРТ (15 МИНУТ)
================================================================================

1. TELEGRAM БОТ (2 мин)
   → Найти @BotFather в Telegram
   → /newbot → получить BOT_TOKEN

2. GOOGLE CLOUD (3 мин)
   → console.cloud.google.com → новый проект
   → Включить Google Sheets API
   → Включить Google Drive API
   → Создать Service Account → скачать credentials.json

3. GOOGLE SHEETS (1 мин)
   → sheets.google.com → новая таблица
   → Скопировать ID из URL

4. ВЫДАЧА ДОСТУПА (2 мин)
   → Открыть credentials.json → скопировать client_email
   → Google Sheets → Поделиться → вставить email → Редактор

5. КОНФИГУРАЦИЯ (2 мин)
   → Создать .env:
      BOT_TOKEN=your_token
      SPREADSHEET_ID=your_id

6. УСТАНОВКА (3 мин)
   → pip install -r requirements.txt

7. ЗАПУСК (2 мин)
   → python test_local.py (проверить конфиг)
   → python bot.py (запустить бот)

8. ПРОВЕРКА (1 мин)
   → Telegram → найти бота → /start → готово!
